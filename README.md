# Smart Daily Planner

A complete and practical daily planner app designed to help users manage tasks, stay consistent, and track their productivity over time.

## Features

- Create, edit, and manage daily tasks
- Self-scoring system to evaluate task completion
- Reminder system for important future events (e.g., job interviews, deadlines)
- Weekly and monthly progress tracking
- Visual charts for productivity analysis
- Clean and user-friendly interface

## How It Works

- Users define their daily tasks
- After completing tasks, they assign a score to themselves
- The app tracks performance over time
- Charts provide insights into weekly and monthly progress
- Reminders help users stay aware of important upcoming events

## Tech Stack

- Flutter (mobile application)
- Local database / storage
